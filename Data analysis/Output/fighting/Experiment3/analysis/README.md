# README

## Overview

This directory contains analysis results for fighting behavior during Experiment 3. It includes a bar chart visualizing circling behavior, correlation analysis results, and a CSV file containing merged frame data.

### Module Role

This directory is a subdirectory of `SecurePig\Data analysis\Output\fighting` and contains the analysis results specific to Experiment 3.

## Directory and File Details

This directory contains the following files:

-   `circlingBehaviourBarChart.png`: A bar chart visualizing circling behavior during fights.
-   `correlationAnalysis.txt`: A text file containing the results of correlation analysis performed on the fighting data.
-   `mergedFrame.csv`: A CSV file containing merged frame data, including the following columns:
    -   `Day`: The day of the experiment.
    -   `AnimalID`: The ID of the animal.
    -   `Score`: A score related to fighting behavior.
    -   `CirclingScore`: A score related to circling behavior.
    -   `ScoreDiff`: The difference in score compared to the previous day.
    -   `CirclingDiff`: The difference in circling score compared to the previous day.
